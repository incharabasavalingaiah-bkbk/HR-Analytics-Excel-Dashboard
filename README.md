# HR Analytics Dashboard (Excel)

## 📌 Project Overview
This project is an HR Analytics Dashboard created in Microsoft Excel to analyze employee data and provide insights into workforce trends. It helps HR professionals understand employee distribution, attrition, and salary patterns using interactive charts and Pivot Tables.

## 🎯 Objectives
- Analyze employee data efficiently.
- Identify department-wise employee distribution.
- Monitor employee attrition.
- Compare average monthly income across departments.
- Visualize gender distribution.
- Analyze monthly income by job role.

## 📊 Dashboard Features
- Employee Count by Department
- Attrition by Department
- Gender Distribution
- Average Monthly Income by Department
- Monthly Income by Job Role

## 🛠️ Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Data Cleaning
- Dashboard Design

## 📂 Dataset
IBM HR Employee Attrition Dataset
